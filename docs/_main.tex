\hypertarget{documentation-notes}{%
\section{Documentation notes}\label{documentation-notes}}

There are various types of documentation throughout the WildlifeSystems project, each intended for a different audience.

\begin{itemize}
\item
  \textbf{Code Comments} are throughout the source code for the project. These explain how the underlying computer code works for programmers.
\item
  \textbf{Package Documentation} explains the functionality for each individual package. The intended audience for these is those implementing sensor networks, and end users wishing to develop a deeper technological understanding of the system.
\item
  \textbf{Manuals} such as this which provide a high level overview suitable for end users.
\end{itemize}
